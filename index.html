What is this?

news.brain.run is an experiment in which I have run a data-orchestration pipeline against the arVix.org feed of the latest articles published in the new (November 2023) Accessible HTML format and have feed this pipeline into a number of bespoke LLM summarization chains. I then feed these into a duckdb database which is pulled into this display by streamlit.io.

Basically I gathered a bunch o' academic articles, threw a robot at it to summarize it for me, then published those summaries here. The robots take a bit to figure out these articles because I ask them a lot of questions, so I set it up so the robots can do this stuff in the background. All of this is pretty inexpensive and I hope to make a cheap little robot for anyone to learn how stuff works.
